# Changelog

## [1.0.0] - 2026-05-16

### Added
- Initial release
- Paste images from clipboard directly into VSCode terminal
- Support for screenshots and copied image files
- Works on Windows, Linux, and macOS
- Automatic WSL path conversion for Windows users
- Customizable save directory via settings
- Interactive filename prompt with timestamped defaults
- Support for multiple image formats (PNG, JPG, JPEG, GIF, BMP, WebP, SVG, ICO, TIFF)
- Keyboard shortcut: Ctrl+Shift+Alt+I (when terminal is focused)
- Command palette integration: "Paste Image into Terminal"
